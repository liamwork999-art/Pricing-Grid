# 💳 Pricing Grid

A modern pricing section built with HTML and CSS. This project showcases a clean comparison layout for different plans such as Free, Pro, and Enterprise.

## ✨ What this project does

This project highlights:

- responsive card-based pricing layout
- clean typography and alignment
- button styling and call-to-action design
- simple but professional UI composition

## 🧱 What the page includes

The pricing grid contains three plan cards:

- 🆓 Free
- ⭐ Pro
- 🏢 Enterprise

Each card includes:

- a plan name
- a price
- a short description
- feature highlights
- a button to get started

## 🎯 Purpose

This kind of layout is commonly used in landing pages and SaaS websites. It is a great project for practicing structure, spacing, colors, and card styling.

## ▶️ How to run

1. Open the project in VS Code.
2. Open [Pricing_Grid/index.html](Pricing_Grid/index.html).
3. Preview it with Live Server or open the file in a browser.
